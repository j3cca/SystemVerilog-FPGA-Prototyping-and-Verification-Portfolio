# SystemVerilog FPGA Prototyping and Verification Portfolio
A collection of synthesizable SystemVerilog modules and self-checking testbenches developed in Vivado for use with the CMOD S7.
* The CMOD S7 is a breadboardable FPGA that utilizes Xilinx's Spartan 7 Chip.
* Because the CMOD S7 is a DIP-format FPGA module, I built a custom peripheral interface on a solderless breadboard, including DIP switches, indicator LEDs, current-limiting resistors, and pull-down networks mapped to the XDC pin constraints in Vivado.

Developed by Jessica Milner during Summer 2026 using "FPGA Prototyping by SystemVerilog Examples" by Pong P. Chu and "SystemVerilog for Verification" by Chris Spear and Greg Tumbush.
